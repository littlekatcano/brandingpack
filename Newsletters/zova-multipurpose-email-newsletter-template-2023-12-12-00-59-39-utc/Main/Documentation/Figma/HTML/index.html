C) Basic Guidelines for Editing in FIGMA Template - top

Here are some basic steps for editing in this FIGMA template. First of all I recommend to installing fonts to your system font folder before editing the FIGMA template. I also recommend to use Dekstop use of FIGMA as some font not recogized by FIGMA through browser. Font name and download link mentioned here

Editing the text or logo:



Open zova.fig file in FIGMA.

Double Click on particulared text elements and start editng it & replace image or logo by just drag and drop it.

Note:Follow the same steps for any other text editing also.

---

How to Change Color:

Step 1: First double click on particulared elements to change color

 

Step 2-3: Than click on "Fill" option in right panel. click on Color Pallette Mode as mentioned below:

 

Step 4: Choose the color what you want and than click on color, its done :)

 

Change color for any elements is very easy for text, icon or background shape.

Note:You can follow the same steps for any other elements like text, icon etc.

---

How to place images using drag-n-drop :

You can easily place any image or screenshot of your work in FIGMA template using just drag-n-drop. example given below :


Step1
Go to particular layer or group by continue double click on it, "Choose Image" panel will open..

Step 2
Now choose image from your computer which you want to replace or place here..

Step3
than just drag it .. on Panel "Choose Image".. and its done.. :)
